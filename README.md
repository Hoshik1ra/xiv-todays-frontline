# XIV Today Frontline

A Vite + TypeScript static app for GitHub Pages that shows today's Final Fantasy XIV Frontline campaign.

Target GitHub Pages URL:

https://Hoshik1ra.github.io/xiv-todays-frontline/

## Stack

- Vue Vapor with TSX through `vue-jsx-vapor`
- `vue-i18n` for `zh-CN` and `en-US`
- `mdui` Material Design 3 web components
- Tailwind CSS
- Day.js

## Commands

```bash
pnpm install
pnpm dev
pnpm build
```

The Vite base path is configured as `/xiv-today-frontline/` for GitHub Pages.
